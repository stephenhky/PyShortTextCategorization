Tutorial
========

After installation, you are ready to start testing the convenience and power
of the package.

Before using, type

>>> import shorttext

You will get the message that `Theano` or `Tensorflow` backend is used for `keras`. Refer to `Keras Backend
<https://keras.io/backend/>`_ for information about switching backends.
